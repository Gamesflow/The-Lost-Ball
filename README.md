
# The Lost Ball

A 3D runner game inspired from Subway Surfers and built on ThreeJS. 

## How to play 

|Key| Controls |
|--|--|
| Spacebar/Up arrow | Jump |
| D/Right Arrow | Go Right |
| A/Left Arrow | Go left |
